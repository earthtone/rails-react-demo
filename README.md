# Stuff to install (if needed)

1. https://postgresapp.com/
2. gem install rails
3. gem install foreman
4. npm i yarn

# Setup
1. cd into directory
2. bundle install
3. yarn install
4. rake db:migrate 

# Start Development Service
1. foreman start -f Procfile.dev <= should start both rails and javascript server
2. go to http://localhost:3000/
